When I was deciding how to build this site, I wanted something that:

1. Loads fast by default
2. Doesn't force me into a heavy JS framework for static pages
3. Plays nicely with Markdown for a blog
4. Deploys easily to GitHub Pages

Astro checked all of these boxes. Pages ship as static HTML with
basically zero JavaScript unless I explicitly add it, and the content
collections API makes managing blog posts straightforward.
